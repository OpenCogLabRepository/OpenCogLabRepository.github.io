The Rapid Visual Information Processing Task

This is a PsychoPy (Peirce, 2008) implementation of the rapid visual information processing (RVIP) task described in Wesnes and Warburton, 1983 and is similar to the implementation provided with Cambridge Cognitionâ€™s CANTAB cognitive battery (Sahakian and Owen, 1992).

The RVIP assesses working memory and sustained attention. Neuroimaging experiments using PET have found that this task activates a bilateral fronto-parietal network (Coull et al., 1996). Response times and detection accuracy from this task have been previously correlated with DMN function (Pagnoni, 2012).

See the repository README for more information.
